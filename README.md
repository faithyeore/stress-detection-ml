# DSA4262 Stress Detection Project

This repository contains a reproducible Jupyter Notebook pipeline for stress detection in social media text.
Models evaluated include a keyword baseline, traditional ML (e.g. Logistic Regression / SVM), and BERT.

## Repository Structure
- `notebooks/` – main notebook(s)
- `src/` – optional helper scripts (if any)
- `requirements.txt` – Python dependencies
- `data/` – dataset location (not included if large)
- `reports/` – figures / exported PDF (optional)
